## 画像サイズの推奨

### サムネイル画像（作品一覧用）
- 推奨サイズ: 400x300px
- フォーマット: PNG, JPG, WebP
- 最適化: 適切な圧縮でファイルサイズを小さく保つ

### 詳細画像（作品詳細用）
- 推奨サイズ: 800x600px 以上
- フォーマット: PNG, JPG, WebP
- 品質: 高品質を維持しつつ適切に圧縮

## 注意事項

1. 画像ファイル名は半角英数字で統一してください
2. 大文字・小文字は区別されます
3. 画像の著作権に注意してください
4. 適切なaltテキストをHTMLに設定してください
5. レスポンシブ対応を考慮した画像サイズを使用してください

## プレースホルダー画像

現在はプレースホルダーとして、実際の画像ファイルを準備してください。
実際のデプロイ前にすべての画像ファイルをこのフォルダに配置する必要があります。
